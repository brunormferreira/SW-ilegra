# SWAPI-ilg

- This project intends to use [SWAPI API](https://swapi.co/)

## Running
    
   - On command line do as follow:
   
   ```
    $ git clone https://github.com/brunormferreira/swapi-ilg.git
    $ cd swapi
    $ git checkout dev
    $ yarn/npm install
    $ yarn/npm start
   ```
